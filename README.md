# Osoyoo Line Following Robot

## Overview
This project involves the design and implementation of a line-following robot using C++. The robot was built to detect and track a line path using sensor input and motor control logic. Through iterative improvements in the code, the system achieved consistent performance and reliable navigation.

## Key Achievements
- Achieved consistent line-tracking performance across multiple runs
- Improved sensor handling and motor control through iterative code refinement
- Reduced completion time across attempts, reaching a fastest time of **27 seconds**
- Achieved a **100% task completion rate**
- Placed **1st in Lab 01 Competition 1**

## Features
- Real-time line detection using sensors
- Motor control for smooth movement and directional correction
- Adaptive adjustments to handle sharp and soft turns
- Stable performance across different track conditions

## Technologies Used
- C++
- Arduino IDE / Embedded programming environment (if applicable)
- Osoyoo Robot Kit hardware

## System Design
The robot continuously reads sensor inputs to detect the line and adjusts motor speed and direction accordingly. Iterative improvements focused on improving responsiveness during turns and reducing tracking errors.

## Performance Summary
- Best completion time: **27 seconds**
- High reliability across multiple test runs
- Competitive performance in lab environment

## Purpose
The goal of this project was to develop an efficient and reliable line-following system while improving real-time decision-making through embedded programming and sensor integration.
